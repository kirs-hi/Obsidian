# 别再饿瘦：要“吃对”

160斤男生外卖减脂30天
核心 = 热量缺口300-500kcal
+ 高蛋白优先
目标：掉脂不掉肌

---

# 80kg营养目标（照这个区间）

蛋白质：128-160g/天
脂肪：48-64g/天
碳水：训练前后优先吃
饮水：2L起，训练日2.5L
先执行14天再微调

---

# 三餐照抄模板（外卖版）

早餐：2个水煮蛋 + 牛奶1瓶
午餐：瘦肉主菜 + 双份蔬菜 + 半份主食
晚餐：少主食 + 蛋白质 + 蔬菜
每餐吃到7-8分饱

---

# 点单公式 + 避坑清单

公式：高蛋白主菜1份
+ 蔬菜2份 + 主食半份 + 0糖饮料
避坑1：重油重酱
避坑2：奶茶酒精
避坑3：夜宵零食

---

# 周末大餐不反弹 SOP

顺序：先蛋白 → 再蔬菜 → 最后主食
第二天直接恢复正常减脂餐
不要补偿性断食
连续做满30天
评论“外卖清单”领关键词+菜单
