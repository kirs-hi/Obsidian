# 160斤男生先记住这条公式

热量缺口 300-500kcal
蛋白质优先，别硬饿

# 三餐照抄（80kg参考）

早餐：2个水煮蛋 + 1瓶牛奶
午餐：瘦肉 + 双份蔬菜 + 半碗米饭
晚餐：少主食 + 蛋白质 + 蔬菜

# 外卖点单万能公式
